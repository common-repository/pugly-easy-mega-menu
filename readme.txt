== Description ==

[youtube https://www.youtube.com/watch?v=V7FbXDoDdag]

An easy to use plugin to easily create super awesome menus that include search, shopping cart and easy mobile menus.

1. Easily insert menu items, carts and search
2. Interactive link selector
3. Quickly style an item as a button
4. Option to pin menu item on mobile so they so after mobile break
5. One-click centered logo
6. Multi-Column dropdown
7. Top Bar Menu
8. Auto mobile menu
9. Define breakpoint
10. Intuitive and easy interactive styling
11. 100% FREE

== Installation ==

1. Install easily with the WordPress plugin control panel or manually download the plugin and upload the folder to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Create or edit a menu by clicking the "Pugly Menu" admin menu link
4. Style a menu by clicking the "Pugly Style" admin menu link
